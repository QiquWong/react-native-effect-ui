effect-ui write with react-native
========================

### Requirements, install as needed: ###

##### React Native:#####
```
$ npm i -g react-native-cli
```

##### watchman:#####
```
$ brew install watchman
```

##### clone this repo: #####

```
$ cd react-native-effect-ui
$ npm install
$ ./build.sh
```
